# Kubeadm::Join::Helper

## Installation

    $ bundle --binstubs=bin

## Usage

### Create new token on master node and upload to s3

    $ bin/kubeadm-join-helper -u

### Join worker node

    $ bin/kubeadm-join-helper -j
